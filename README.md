# PWr Quantum Computing Labs

This repository contains materials and assignment work for the PWr quantum computing labs. The project is based on Jupyter notebooks and focuses on numerical and introductory quantum computing exercises using Python libraries such as NumPy, SciPy, and related notebook tooling.

## Repository Contents

- `intro_numpy_qiskit_lab.ipynb` - introductory lab material
- `lab1_numpy_quantum_states_student.ipynb` - Lab notebook
- `assignments/` - submitted assignment notebooks organized by lab

## Assignment Submission Rules

For each assignment submission:

1. Create a feature branch and make a copy of the lab `X` notebook in `assignments/lab_X/`.
2. Name the notebook `firstname_lastname_lab_X.ipynb`.
3. Create a pull request and assign the instructor as a reviewer.
